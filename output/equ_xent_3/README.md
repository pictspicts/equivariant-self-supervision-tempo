# 学習録: 超高速ハイブリッドモデル + GPU無劣化ストレッチ + GradNorm (最終完全版)

## 現在の設定
* **実験名**: `equ_xent_3`
* **Loss設定**: `hybrid` (等変性損失 ＋ シフト交差エントロピー正則化)
* **データ拡張手法**: GPU画像ベース無劣化ストレッチ (Bilinear) ＋ 並進不変性（Translation-Invariance）獲得ランダムクロップ
* **重み調整手法**: PESTO互換 勾配ベース動的ロスバランシング (Gradients Loss Weighting / GradNorm)
* **バッチサイズ**: 64

## 到達した究極の学習パラダイム

### 1. カンニングの根絶と「超高速化」の完全両立 (CPU → GPUへの解放)
旧モデルの最大のボトルネックであった「データローダー（CPU）によるSoXフェーズボコーダ処理」を完全に無効化し、データ装填のレイテンシを事実上ゼロにしました。代わりに、ディープ・ネットワーク直前の `Frontend` (GPU) において PyTorch ネイティブの `F.interpolate (bilinear)` による二次元画像リサイズ処理を実行しています。
* **無劣化と超高速**: 音声のシュワシュワ音（アーティファクト）が消滅し、AIへのカンニング・ヒントを完全に遮断。さらに、シングルコアCPUによる重い波形計算が数千コアのCUDA行列計算に置き換わったため、体感で数倍〜数十倍の学習速度向上が実現しました。
* **スーパーサンプリング＋クロップ**: 長めに読み込んだ画像（17.6秒相当）をストレッチした後、必ず「無音が一切含まれない長さ（13.6秒）」をランダムに窓抜きします。これにより、パディング漏洩を防ぎつつ「リズムの位置ズレ耐性（並進不変性）」をモデルに付与しています。

### 2. 勾配ベースの動的重み調整 (Gradients Loss Weighting / GradNorm)
PESTO本家で証明されている最先端の学習バランス手法を完全に移植しました。
* `torch.autograd.grad(retain_graph=True)` を用いて「それぞれのLossがネットワーク最終層の重みをどれだけの強さで更新しようとしているか（L2ノルム）」をリアルタイム観測。
* 各バッチ毎に、強すぎるロスのウェイト（$w$）を減衰させ、弱まっているロスを支援する「自動等量分配機能」が働きます。これにより「等変性」と「不変性」が常に 1:1 の完全な影響力で綱引きを行い、学習のモード崩壊を論理的に防ぎます。

### 3. PESTO完全互換の数式構造
PESTOの $z_2 / z_1$ によるスケール崩壊防止策を、「対数空間におけるHuber差異」として最適化して実装しています。また、GPU側でランダム生成されたストレッチ倍率（`ts_rate`）を直接ロス関数のターゲットとして流し込む完璧な連携（Coupling）を実現しており、数学上の死角はありません。
