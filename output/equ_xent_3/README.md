# 学習録: Hybrid Loss + 無劣化タイムストレッチ + GradNorm (最終形態)

## 現在の設定
* **実験名**: `equ_xent_3`
* **Loss設定**: `hybrid` (等変性損失 ＋ シフト交差エントロピー)
* **データ拡張手法**: 無劣化画像ストレッチ ＋ 並進不変性（Translation-Invariance）獲得ランダムクロップ
* **重み調整手法**: PESTO互換 勾配ベース動的ロスバランシング (Gradients Loss Weighting / GradNorm)
* **バッチサイズ**: 64

## 全ての問題をクリアした「最終形態」の実装ポイント

### 1. 完璧なカンニング対策 (無劣化処理 + スーパーサンプリング)
以前の実験でAIが「ノイズ」や「無音パディングの量・切断面」を見てタイムストレッチの倍率を推測し、一定のテンポばかりを出力してズルをしていた現象を完全に封じ込めました。
* **無劣化ストレッチ**: 音声のフェーズボコーダ処理を完全廃止し、対数メルスペクトログラム状態にした後、画像処理である線形補間 (`F.interpolate`) を用いた数学的な時間軸リサイズに移行しました。
* **スーパーサンプリング**: データローダーで規定より長い1.3倍の長さ（780,000サンプル）を読み込みます。
* **ランダムクロップ**: ストレッチ後の巨大な画像から、「無音パディングが1ミリも含まれないピッタリ規定サイズ（1361フレーム）のテンポ窓」をランダム位置からクロップ（くり抜き）します。これによりAIは縮尺のヒントを完全に失い、併せて自己教師あり学習に必須の **「並進不変性（Translation-Invariance）」**（切り取る場所がズレてもテンポは不変であるという堅牢さ）も獲得しました。

### 2. 勾配ベースの動的重み調整 (Gradients Loss Weighting / GradNorm)
PESTO本家で採用されている最先端の学習バランス手法を完全に移植・再現しました。単なるLoss値同士の割り算ではなく、**「それぞれのLossがニューラルネットワーク（予測最終層）の重みを更新しようとする『力』の強さ（勾配＝L2ノルム）」** を直接取得 (`torch.autograd.grad`) し、学習パワーのバランスを統制します。
* どのような楽曲・状況にあっても、必ず「等変性ロスの学習力」と「正則化ロスの学習力」が常に均等な影響力になるように、自動的に重み（ウェイト $w_{eq}$, $w_{xe}$）を算出してコントロールします。これにより片方のロスが強すぎて学習が崩壊する事態を防ぎます。

### 3. PESTO完全互換の数式構造 (スケール不変ロス)
等変性のロジックにおいて、PESTOが行っている「確率圧縮値の割り算を用いたスケール崩壊防止策（`huber(f2/f1, ratio)`）」を、対数空間での減算（`huber(log(phi_2) - log(phi_1), k*log(alpha))`）として数学的等価な状態で完璧に翻訳・最適化し実装しています。シフトエントロピーの移動幅も `log_bin_width` に基づく完璧な平行移動（Roll）として動作します。
